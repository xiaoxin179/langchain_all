# 1. API 密钥管理模块

## 需求概述
编写一个独立的配置文件用于存储和管理大模型 API 密钥，避免在代码中硬编码敏感信息，确保密钥安全性和项目可维护性。

## 功能需求

### 核心功能
1. **密钥存储**：将各平台 API 密钥存储在项目根目录的 `.env` 文件中
2. **密钥读取**：提供统一函数接口，按需获取指定密钥
3. **自动加载**：应用启动时自动加载 `.env` 文件到环境变量

### 支持的密钥类型
- `DEEPSEEK_API_KEY` — DeepSeek 模型密钥
- `OPENAI_API_KEY` — OpenAI 模型密钥
- （可扩展其他平台）

## 输出文件结构

```
langchain_chat/
├── .env              # API 密钥存储文件（需加入 .gitignore）
├── config.py         # 密钥管理模块
└── ...
```
